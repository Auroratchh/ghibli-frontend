# ghibli-frontend

Interfaz de usuario de la aplicación Studio Ghibli. 
Consume el backend propio para mostrar películas y personajes de la API de Studio Ghibli.

## Tecnologías
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Variables de entorno
```env
NEXT_PUBLIC_BACKEND_URL=http://localhost:3001
```

## Cómo correrlo
```bash
npm install
npm run dev
```
Abre http://localhost:3000 en el navegador.
